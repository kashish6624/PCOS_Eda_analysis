# PCOS Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a Polycystic Ovary Syndrome (PCOS) dataset to identify patterns and relationships between clinical, and hormonal indicators associated with PCOS diagnosis.

The objective of this analysis is to understand key factors linked with PCOS and derive meaningful insights through statistical exploration and visualization.

---

## Dataset Description
The dataset contains patient-level medical information including:

- Age
- Body Mass Index (BMI)
- Testosterone Levels (ng/dL)
- Menstrual Cycle Regularity
- Antral Follicle Count
- PCOS Diagnosis (Target Variable)

Each row represents an individual patient record.

## Exploratory Data Analysis Steps
1. Data Understanding and Structure Inspection
2. Data Quality Checks (Missing Values & Duplicates)
3. Univariate Analysis
4. Outlier Detection
5. Feature Relationship Analysis
6. Multivariable Analysis
7. Correlation Analysis
8. Key Insight Generation

---

## Key Insights
- Menstrual irregularity shows strong association with PCOS diagnosis.
- Elevated testosterone levels correlate positively with PCOS prevalence.
- Higher antral follicle counts appear strongly linked with PCOS cases.
- BMI demonstrates moderate association but is not a standalone predictor.
- PCOS appears influenced by multiple interacting physiological factors.
